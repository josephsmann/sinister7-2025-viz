# Sinister 7 2025 — Checkpoint Arrivals by Time of Day

Interactive visualization of the 2025 Sinister 7 100-mile relay: every team's
cumulative arrival time at each of the seven checkpoints, on the wall clock,
highlighting **Sadomountainism** and **YEG Harriers**.

Live page → https://josephsmann.github.io/sinister7-2025-viz/

Self-contained (D3 + Observable Plot inlined). Generated from a marimo notebook.
